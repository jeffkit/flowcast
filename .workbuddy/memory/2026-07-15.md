# 2026-07-15

## 修复 GitHub CI 红叉（agentproc 依赖未安装）

- 现象：main（b9e95f62，v0.6 迁移到 agentproc SDK）CI 全红，报
  `ERR_MODULE_NOT_FOUND: Cannot find package 'agentproc'`（executor/agentproc-adapter.js）。
- 根因：`.github/workflows/ci.yml` 和 `publish.yml` 只有 checkout → setup-node → npm test，
  **从未执行 npm install / npm ci**。项目早期零依赖时侥幸能过；v0.6 起引入真实依赖
  `agentproc@^0.10.1` 后必然失败。docs.yml 本来就有 npm install，不受影响。
- 修复（commit 137430b）：
  - ci.yml / publish.yml 各加 `- name: Install dependencies / run: npm ci`
  - setup-node 开启 `cache: npm` 加速
  - 项目已有 package-lock.json，故用 npm ci（可复现）
- 验证：本地 `npm ci` + `node --test --test-concurrency=1 --test-timeout=120000 test/*.test.js`
  → 376 tests pass；推送后 CI 三个矩阵 job（ubuntu×20/22、macos×22）全 success。
